# ELCN-Project

A functional outreach prototype created for the End Lung Cancer Now Association. This project focuses on organizing communication workflows and allowing users to manage email recipients through a simple dashboard interface.

## Features

- Dashboard prototype designed in Figma
- Role-based organization for email management
- Recipient selection for targeted communication
- Working email system for sending messages
- SendGrid integration for email delivery

## Tech Stack

- Python
- HTML
- CSS
- SendGrid
- Figma

## Project Overview

This project was built as a prototype to support nonprofit outreach efforts. The goal was to create a basic but functional web application that helps organize communication and improve the email workflow for users.

## Notes

This project is focused on functionality and workflow rather than final visual design.
